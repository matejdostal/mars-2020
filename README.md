## Sections
##### Arduino
- Code written in ++Arduino++
- Radar, Movement
- Bluetooth communication
- LIDAR senzors
        
##### Server
- ++Node.js++
- Arduino-Server, Server-Client connections
- Polar to Kartesian coordinates
- Serialport, WebSockets
        
##### Client
- ++JavaScript++
- Map creating
- p5.js
- Retrieving data from Server
        
##### Controller
- Sending commands to Server
- Rover movements
